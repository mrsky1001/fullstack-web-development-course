# Урок 00: Подготовка рабочего окружения

## 🎯 Цели урока

После завершения этого урока вы сможете:

- Установить и настроить **Visual Studio Code**
- Установить необходимые **расширения** для веб-разработки
- Создать правильную **структуру проекта**
- Открывать и просматривать файлы в **браузере**
- Использовать **инструменты разработчика** браузера

---

## 📚 Теоретическая часть

### Что такое рабочее окружение?

**Рабочее окружение** (Development Environment) — это набор инструментов, которые программист использует для написания, тестирования и отладки кода.

Для веб-разработки нам понадобятся:

| Инструмент | Назначение |
|------------|------------|
| Редактор кода | Написание HTML, CSS, JavaScript |
| Браузер | Просмотр и тестирование веб-страниц |
| Инструменты разработчика | Отладка и исследование страниц |
| Локальный сервер | Имитация реального веб-сервера |

### Почему Visual Studio Code?

**Visual Studio Code (VS Code)** — это бесплатный редактор кода от Microsoft, который стал стандартом в веб-разработке благодаря:

- **Бесплатность** — полностью бесплатный для любого использования
- **Кроссплатформенность** — работает на Windows, macOS и Linux
- **Расширяемость** — тысячи расширений для любых задач
- **Интеллектуальные подсказки** — автодополнение, подсветка ошибок
- **Интеграция с Git** — встроенная поддержка системы контроля версий
- **Популярность** — используется в большинстве компаний

### Альтернативные редакторы

| Редактор | Плюсы | Минусы |
|----------|-------|--------|
| **WebStorm** | Мощный функционал, умные подсказки | Платный, тяжёлый |
| **Sublime Text** | Быстрый, лёгкий | Платный (условно) |
| **Atom** | Бесплатный, расширяемый | Медленный, закрыт в 2022 |
| **Notepad++** | Простой, бесплатный | Только Windows, мало функций |

## 🔧 Практическая часть

### Шаг 1: Установка Visual Studio Code

1. Перейдите на официальный сайт: [https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Нажмите кнопку **Download for Windows/macOS/Linux**
3. Запустите установщик
4. Следуйте инструкциям мастера установки

> **Рекомендация:** При установке отметьте галочки:
> - "Добавить в контекстное меню проводника"
> - "Зарегистрировать VS Code как редактор по умолчанию"

### Шаг 2: Первый запуск и настройка

После установки откройте VS Code. Вас встретит приветственное окно.

#### Основные настройки

Откройте настройки: `File → Preferences → Settings` или `Ctrl + ,`

Рекомендуемые настройки:

| Параметр | Значение | Зачем |
|----------|----------|-------|
| Auto Save | afterDelay | Автосохранение файлов |
| Font Size | 14-16 | Комфортный размер шрифта |
| Tab Size | 2 | Стандарт для веб-разработки |
| Word Wrap | on | Перенос длинных строк |

### Шаг 3: Установка расширений

Расширения добавляют новые возможности в редактор.

#### Открыть панель расширений:
- Нажмите `Ctrl + Shift + X`
- Или кликните на иконку квадратиков в боковой панели

#### Обязательные расширения:

| Расширение | Описание | Как найти |
|------------|----------|-----------|
| **Live Server** | Локальный сервер с автообновлением | Поиск: "Live Server" |
| **Prettier** | Автоматическое форматирование кода | Поиск: "Prettier" |
| **Auto Rename Tag** | Автопереименование парных HTML-тегов | Поиск: "Auto Rename Tag" |
| **Auto Close Tag** | Автозакрытие HTML-тегов | Поиск: "Auto Close Tag" |

#### Рекомендуемые расширения:

| Расширение | Описание |
|------------|----------|
| **CSS Peek** | Быстрый просмотр CSS-стилей |
| **HTML CSS Support** | Подсказки CSS-классов в HTML |
| **Path Intellisense** | Автодополнение путей к файлам |
| **Material Icon Theme** | Красивые иконки файлов |
| **Russian Language Pack** | Русский интерфейс (опционально) |

### Шаг 4: Создание структуры проекта

Правильная структура проекта — залог удобной работы.

#### Базовая структура веб-проекта:

```
my-project/
│
├── index.html          # Главная страница
├── css/
│   └── style.css       # Основные стили
├── js/
│   └── script.js       # JavaScript код
└── images/             # Изображения
    └── logo.png
```

#### Как создать проект:

1. Создайте папку для проекта (например, `my-first-site`)
2. Откройте VS Code
3. `File → Open Folder` → выберите созданную папку
4. Создайте файлы и папки:
   - Правый клик в панели файлов → `New File` / `New Folder`

### Шаг 5: Открытие проекта в браузере

#### Способ 1: Просто открыть файл (не рекомендуется)
- Дважды кликните на `index.html` в проводнике
- ❌ Минус: нужно вручную обновлять страницу

#### Способ 2: Live Server (рекомендуется)
1. Откройте файл `index.html` в VS Code
2. Правый клик в редакторе → **"Open with Live Server"**
3. Или нажмите кнопку **"Go Live"** внизу справа

✅ Плюсы Live Server:
- Автоматическое обновление при сохранении
- Правильная работа с путями
- Имитация реального веб-сервера

### Шаг 6: Инструменты разработчика браузера

**DevTools** (Инструменты разработчика) — это встроенный в браузер набор инструментов для отладки.

#### Как открыть:
- `F12` или `Ctrl + Shift + I`
- Или правый клик на странице → "Посмотреть код" / "Inspect"

#### Основные вкладки:

| Вкладка | Назначение |
|---------|------------|
| **Elements** | Просмотр и редактирование HTML/CSS |
| **Console** | Вывод сообщений JavaScript, ошибки |
| **Network** | Анализ загрузки ресурсов |
| **Sources** | Просмотр исходного кода, отладка JS |
