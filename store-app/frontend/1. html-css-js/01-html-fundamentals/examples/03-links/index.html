Важные атрибуты тега <a>

  • href — адрес ссылки (обязательный)
  • target — где открыть (_blank, _self, _parent, _top)
  • rel — тип связи (noopener, noreferrer, nofollow)
  • title — подсказка при наведении
  • download — скачивание вместо открытия
