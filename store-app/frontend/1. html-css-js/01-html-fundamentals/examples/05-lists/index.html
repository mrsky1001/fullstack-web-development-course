Нумерованный список (ol)

Простой нумерованный список

  1. Подготовить ингредиенты
  2. Разогреть духовку до 180°C
  3. Смешать все компоненты
  4. Выпекать 30 минут
  5. Дать остыть перед подачей

Атрибуты нумерованного списка

Римские цифры (type="I"):

  1. Первый пункт
  2. Второй пункт
  3. Третий пункт

Маленькие римские (type="i"):

  1. Первый пункт
  2. Второй пункт
  3. Третий пункт

Буквы (type="A" или "a"):

  1. Первый пункт
  2. Второй пункт
  3. Третий пункт

Начать с 5 (start="5"):

  1. Пятый пункт
  2. Шестой пункт
  3. Седьмой пункт

Обратный отсчёт (reversed):

  1. Третье место
  2. Второе место
  3. Первое место!

Лучшие практики использования списков

  1. Выбирайте правильный тип списка:
    • ul — когда порядок НЕ важен
    • ol — когда порядок ВАЖЕН
    • dl — для пар термин-определение
  2. Не злоупотребляйте вложенностью:
    • Оптимально — не более 3 уровней
    • Слишком глубокие списки сложно читать
  3. Используйте списки для навигации:
    • Меню сайта — это семантически список ссылок
    • Оборачивайте nav > ul > li > a
  4. Стилизуйте через CSS:
    • Можно изменить маркеры (list-style-type)
    • Можно использовать свои иконки
