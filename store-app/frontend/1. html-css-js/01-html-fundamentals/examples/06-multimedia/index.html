Форматы медиафайлов

Видеоформаты

Формат Расширение MIME-тип Поддержка
MP4 (H.264) .mp4 video/mp4 Все браузеры ✓
WebM (VP9) .webm video/webm Chrome, Firefox, Edge ✓
Ogg (Theora) .ogv video/ogg Chrome, Firefox ✓
AV1 .mp4 video/mp4; codecs="av01..." Современные браузеры

Формат Расширение MIME-тип Поддержка

Ogg Vorbis .ogg audio/ogg Chrome, Firefox, Edge ✓
WAV .wav audio/wav Все браузеры ✓

Советы по работе с мультимедиа

  1. Всегда указывайте несколько форматов:
    • MP4 + WebM для видео
    • MP3 + OGG для аудио
  2. Используйте poster для видео:
    • Показывает превью до загрузки
    • Улучшает пользовательский опыт
  3. Не используйте autoplay без необходимости:
    • Раздражает пользователей
    • Тратит трафик
    • Блокируется браузерами (со звуком)
  4. Оптимизируйте размер файлов:
    • Сжимайте видео перед загрузкой
    • Используйте современные кодеки (H.265, AV1)
  5. Добавляйте субтитры:
    • Для доступности
    • Для просмотра без звука
