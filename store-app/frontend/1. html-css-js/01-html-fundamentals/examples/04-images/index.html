Типы путей к изображениям

Относительный путь (рекомендуется)

Изображение в папке images:
<img src="images/photo.jpg">

Изображение уровнем выше:
<img src="../logo.png">

Абсолютный путь (внешний ресурс)

Изображение с внешнего сайта:
<img src="https://example.com/image.jpg">

Data URL (встроенное изображение)

Встроенное изображение (Base64):
<img src="data:image/png;base64,...">

Форматы изображений

Формат Расширение Особенности Когда использовать
JPEG .jpg, .jpeg Сжатие с потерями, нет прозрачности Фотографии
PNG .png Без потерь, поддержка прозрачности Логотипы, иконки, скриншоты
GIF .gif Анимация, ограниченные цвета Простые анимации
WebP .webp Современный формат, лучшее сжатие Всё (рекомендуется)
SVG .svg Векторный, масштабируется без потерь Иконки, логотипы, графика
AVIF .avif Новейший формат, лучшее сжатие Когда поддержка достаточна

Советы по оптимизации изображений

  1. Используйте современные форматы — WebP даёт на 25-35% меньший размер файла.
  2. Указывайте размеры — width и height предотвращают прыжки макета.
  3. Используйте lazy loading — loading="lazy" для изображений в нижней части страницы.
  4. Оптимизируйте размер файла — сжимайте изображения перед загрузкой.
  5. Всегда заполняйте alt — это важно для SEO и доступности.
