## 📚 Теоретическая часть

### Что такое HTML?

**HTML (HyperText Markup Language)** — это язык гипертекстовой разметки. Он не является языком программирования, а представляет собой **язык разметки**, который описывает структуру веб-страницы.

> **Аналогия:** HTML — это скелет веб-страницы. Как скелет человека определяет форму тела, так HTML определяет структуру страницы.

### Зачем нужен HTML?

| Функция | Описание |
|---------|----------|
| **Структура** | Определяет заголовки, параграфы, разделы |
| **Семантика** | Придаёт смысл содержимому (это заголовок, это меню, это статья) |
| **Связи** | Создаёт гиперссылки между страницами |
| **Содержимое** | Вставляет текст, изображения, видео, аудио |

### История HTML

| Версия | Год | Особенности |
|--------|-----|-------------|
| HTML 1.0 | 1993 | Первая версия, базовые теги |
| HTML 4.01 | 1999 | Стандартизация, поддержка CSS |
| XHTML | 2000 | Строгий синтаксис на основе XML |
| **HTML5** | 2014 | Современный стандарт, новые теги и API |

**Мы изучаем HTML5** — современный стандарт, поддерживаемый всеми браузерами.

## 🔧 Структура HTML-документа

### Базовый шаблон

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Заголовок страницы</title>
</head>
<body>
    <!-- Содержимое страницы -->
</body>
</html>
```

### Разбор структуры

| Элемент | Назначение |
|---------|------------|
| `<!DOCTYPE html>` | Объявление типа документа (HTML5) |
| `<html>` | Корневой элемент страницы |
| `lang="ru"` | Указание языка содержимого |
| `<head>` | Метаинформация (не отображается) |
| `<meta charset>` | Кодировка символов |
| `<meta viewport>` | Настройки для мобильных |
| `<title>` | Заголовок во вкладке браузера |
| `<body>` | Видимое содержимое страницы |

### Что такое теги?

**Тег** — это специальная метка, которая указывает браузеру, как интерпретировать содержимое.

```html
<tagname>содержимое</tagname>
```

**Виды тегов:**

1. **Парные теги** — имеют открывающий и закрывающий тег:
   ```html
   <p>Это параграф</p>
   <h1>Это заголовок</h1>
   ```

2. **Одиночные теги** — не имеют закрывающего тега:
   ```html
   <br>    <!-- Перенос строки -->
   <hr>    <!-- Горизонтальная линия -->
   <img>   <!-- Изображение -->
   ```

### Атрибуты тегов

**Атрибуты** добавляют дополнительную информацию к тегам:

```html
<тег атрибут="значение">содержимое</тег>
```

**Примеры:**
```html
<a href="https://google.com">Ссылка</a>
<img src="photo.jpg" alt="Описание">
<div id="header" class="container">...</div>
```

## 📝 Основные теги HTML

### Заголовки (h1-h6)

HTML предоставляет 6 уровней заголовков:

```html
<h1>Заголовок 1 уровня</h1>  <!-- Главный заголовок -->
<h2>Заголовок 2 уровня</h2>  <!-- Подзаголовок -->
<h3>Заголовок 3 уровня</h3>
<h4>Заголовок 4 уровня</h4>
<h5>Заголовок 5 уровня</h5>
<h6>Заголовок 6 уровня</h6>  <!-- Наименьший -->
```

> **Важно:** На странице должен быть **только один `<h1>`** — это главный заголовок страницы.

### Параграфы и текст

```html
<p>Это параграф текста.</p>
<p>Это другой параграф.</p>
```

**Форматирование текста:**

| Тег | Назначение | Пример |
|-----|------------|--------|
| `<strong>` | Важный текст (жирный) | `<strong>важно</strong>` |
| `<em>` | Акцент (курсив) | `<em>акцент</em>` |
| `<mark>` | Выделение (маркер) | `<mark>выделено</mark>` |
| `<del>` | Удалённый текст | `<del>удалено</del>` |
| `<ins>` | Вставленный текст | `<ins>добавлено</ins>` |
| `<sub>` | Подстрочный | H`<sub>2</sub>`O |
| `<sup>` | Надстрочный | E=mc`<sup>2</sup>` |

### Ссылки

```html
<!-- Внешняя ссылка -->
<a href="https://google.com">Перейти в Google</a>

<!-- Внутренняя ссылка -->
<a href="about.html">О нас</a>

<!-- Ссылка в новой вкладке -->
<a href="https://google.com" target="_blank">Открыть в новой вкладке</a>

<!-- Ссылка на email -->
<a href="mailto:email@example.com">Написать письмо</a>

<!-- Ссылка на телефон -->
<a href="tel:+79991234567">Позвонить</a>

<!-- Якорная ссылка (на элемент страницы) -->
<a href="#section1">Перейти к разделу</a>
```

### Изображения

```html
<img src="путь/к/изображению.jpg" alt="Описание изображения">
```

**Обязательные атрибуты:**
- `src` — путь к файлу изображения
- `alt` — альтернативный текст (для SEO и доступности)

**Дополнительные атрибуты:**
```html
<img 
    src="photo.jpg" 
    alt="Моё фото" 
    width="300" 
    height="200"
    title="Подсказка при наведении"
>
```

### Списки

**Маркированный список (неупорядоченный):**
```html
<ul>
    <li>Первый элемент</li>
    <li>Второй элемент</li>
    <li>Третий элемент</li>
</ul>
```

**Нумерованный список (упорядоченный):**
```html
<ol>
    <li>Шаг первый</li>
    <li>Шаг второй</li>
    <li>Шаг третий</li>
</ol>
```

**Вложенные списки:**
```html
<ul>
    <li>Фрукты
        <ul>
            <li>Яблоки</li>
            <li>Бананы</li>
        </ul>
    </li>
    <li>Овощи
        <ul>
            <li>Морковь</li>
            <li>Капуста</li>
        </ul>
    </li>
</ul>
```

### Разрывы и линии

```html
<!-- Перенос строки -->
Первая строка<br>
Вторая строка

<!-- Горизонтальная линия -->
<hr>
```

## 🧱 Блочные и строчные элементы

### Блочные элементы

**Занимают всю ширину** родителя и начинаются с новой строки.

```html
<div>Универсальный блочный контейнер</div>
<p>Параграф</p>
<h1>Заголовок</h1>
<ul>Список</ul>
```

### Строчные элементы

**Занимают только необходимую ширину** и располагаются в строке.

```html
<span>Универсальный строчный контейнер</span>
<a>Ссылка</a>
<strong>Жирный текст</strong>
<em>Курсив</em>
<img>Изображение</img>
```

### Наглядное сравнение

```
Блочные элементы:
┌────────────────────────────────────┐
│ <div>Занимает всю строку</div>     │
└────────────────────────────────────┘
┌────────────────────────────────────┐
│ <p>Каждый с новой строки</p>       │
└────────────────────────────────────┘

Строчные элементы:
┌──────────────────────────────────────────────────┐
│ <span>Один</span> <span>Два</span> <a>Три</a>    │
└──────────────────────────────────────────────────┘
```

## 🖼 Мультимедиа

### Видео

```html
<video controls width="640" height="360">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Ваш браузер не поддерживает видео.
</video>
```

**Атрибуты video:**
- `controls` — показать элементы управления
- `autoplay` — автовоспроизведение
- `loop` — зацикливание
- `muted` — без звука
- `poster` — изображение-превью

### Аудио

```html
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Ваш браузер не поддерживает аудио.
</audio>
```

## 📖 Глоссарий

| Термин | Определение |
|--------|-------------|
| **HTML** | HyperText Markup Language — язык гипертекстовой разметки |
| **Тег** | Специальная метка в угловых скобках `< >` |
| **Атрибут** | Дополнительная информация внутри открывающего тега |
| **Элемент** | Тег вместе с его содержимым |
| **DOM** | Document Object Model — объектная модель документа |
| **Вложенность** | Размещение элементов внутри других элементов |
| **Блочный элемент** | Элемент, занимающий всю ширину и начинающийся с новой строки |
| **Строчный элемент** | Элемент, занимающий только необходимую ширину |
