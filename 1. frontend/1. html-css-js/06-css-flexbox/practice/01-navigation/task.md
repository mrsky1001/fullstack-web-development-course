# Практическое задание: Навигационное меню

## 🎯 Цель

Создать адаптивное навигационное меню с помощью Flexbox.

---

## 📋 Задание

### Десктоп (> 768px):

```
┌──────────────────────────────────────────────────┐
│ LOGO          Главная  О нас  Услуги  Контакты  │
└──────────────────────────────────────────────────┘
```

### Мобильные (< 768px):

```
┌─────────────────────────────┐
│ LOGO                    ☰   │
├─────────────────────────────┤
│ Главная                     │
│ О нас                       │
│ Услуги                      │
│ Контакты                    │
└─────────────────────────────┘
```

### Требования:

1. На десктопе: лого слева, ссылки справа
2. На мобильных: "бургер" меню
3. Плавные hover-эффекты
4. Активная страница выделена
5. Фиксированная позиция (sticky)

---

## ✅ Критерии

- [ ] Flexbox для раскладки
- [ ] Адаптивность (медиа-запрос)
- [ ] Hover-эффекты на ссылках
- [ ] Бургер-меню (иконка)
- [ ] Sticky позиция

---

## 💡 Подсказка

```css
.navbar {
    display: flex;
    justify-content: space-between;
    align-items: center;
    position: sticky;
    top: 0;
}

.nav-links {
    display: flex;
    gap: 20px;
}

@media (max-width: 768px) {
    .nav-links {
        display: none;
        flex-direction: column;
    }
    
    .nav-links.active {
        display: flex;
    }
}
```
