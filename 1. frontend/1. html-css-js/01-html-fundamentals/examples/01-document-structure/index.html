Структура HTML-документа

Это пример правильной структуры HTML-документа. Каждая веб-страница должна содержать все обязательные элементы, показанные в этом примере.

Обязательные элементы

  • <!DOCTYPE html> — объявление типа документа
  • <html> — корневой элемент
  • <head> — метаинформация
  • <body> — содержимое страницы

Откройте исходный код этой страницы (Ctrl+U) и изучите комментарии.
