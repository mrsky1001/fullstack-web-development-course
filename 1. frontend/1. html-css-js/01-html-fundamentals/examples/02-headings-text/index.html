Параграфы текста

Это первый параграф. Параграфы используются для разделения текста на логические блоки. Браузер автоматически добавляет вертикальные отступы между параграфами.

Это второй параграф. Обратите внимание, что между параграфами есть пустое пространство. Это стандартное поведение браузера.

Даже если в коде текст написан на разных строках, браузер отобразит его как одну строку.

Форматирование текста

Важный текст выделяется жирным шрифтом.

Акцентированный текст выделяется курсивом.

Выделенный текст привлекает внимание.

Старая цена: 1000 рублей

Новая цена: 799 рублей

Формула воды: H2O

Формула Эйнштейна: E = mc2

Чисто визуальные теги (устаревший подход)

Жирный текст (тег b) — только визуальное оформление.

Курсивный текст (тег i) — только визуальное оформление.

Подчёркнутый текст — избегайте, может путать со ссылками.

Комбинирование тегов

Это жирный и курсивный текст одновременно.

Это выделенный и важный текст.
