## 📚 Теоретическая часть

### Что такое CSS?

**CSS (Cascading Style Sheets)** — каскадные таблицы стилей. Это язык для описания внешнего вида HTML-документа.

| Технология | Отвечает за |
|------------|-------------|
| **HTML** | Структура и содержимое |
| **CSS** | Внешний вид |
| **JavaScript** | Поведение и интерактивность |

> **Аналогия:** Если HTML — это скелет дома, то CSS — это отделка: покраска стен, укладка пола, выбор штор.

### История CSS

| Версия | Год | Ключевые возможности |
|--------|-----|----------------------|
| CSS1 | 1996 | Базовые стили: шрифты, цвета, отступы |
| CSS2 | 1998 | Позиционирование, z-index, media types |
| CSS2.1 | 2011 | Исправления и уточнения |
| **CSS3** | 2011+ | Модульный подход, анимации, Flexbox, Grid |

**Мы изучаем CSS3** — это не единый стандарт, а набор модулей, которые развиваются независимо.

## 🔌 Способы подключения CSS

### 1. Внешний файл (рекомендуется)

```html
<head>
    <link rel="stylesheet" href="css/style.css">
</head>
```

**Плюсы:**
- Разделение HTML и CSS
- Переиспользование между страницами
- Кэширование браузером

**Минусы:**
- Дополнительный HTTP-запрос

### 2. Внутренние стили (в `<style>`)

```html
<head>
    <style>
        body {
            background-color: #f0f0f0;
        }
        h1 {
            color: blue;
        }
    </style>
</head>
```

**Когда использовать:**
- Критические стили "above the fold"
- Уникальные стили для одной страницы
- Email-рассылки

### 3. Инлайн-стили (атрибут style)

```html
<p style="color: red; font-size: 18px;">Текст</p>
```

**Когда использовать:**
- Динамические стили через JavaScript
- Быстрое тестирование

**Минусы:**
- Смешивание HTML и CSS
- Невозможно переиспользовать
- Высокая специфичность

## 📝 Синтаксис CSS

### Структура правила


**Пример:**

```css
h1 {
```

## 🎯 Селекторы CSS

### Базовые селекторы

| Селектор | Пример | Что выбирает |
|----------|--------|--------------|
| Элемент | `p` | Все `<p>` |
| Класс | `.card` | Всё с `class="card"` |
| ID | `#header` | Элемент с `id="header"` |
| Универсальный | `*` | Все элементы |

### Примеры

```css
/* Селектор элемента */
p {
    line-height: 1.6;
}

/* Селектор класса (начинается с точки) */
.highlight {
    background-color: yellow;
}

/* Селектор ID (начинается с #) */
#main-title {
    font-size: 48px;
}

/* Универсальный селектор */
* {
    margin: 0;
    padding: 0;
}
```

### Комбинированные селекторы

```css
/* Несколько классов */
.card.featured {
    border: 2px solid gold;
}

/* Элемент с классом */
p.intro {
    font-size: 1.2em;
}

/* Группировка (через запятую) */
h1, h2, h3 {
    font-family: 'Georgia', serif;
}
```

### Селекторы отношений

```css
/* Потомок (любой вложенности) */
article p {
    text-indent: 20px;
}

/* Дочерний (прямой потомок) */
ul > li {
    list-style: none;
}

/* Соседний (сразу после) */
h2 + p {
    font-weight: bold;
}

/* Все соседние (все после) */
h2 ~ p {
    color: gray;
}
```

### Селекторы атрибутов

```css
/* Наличие атрибута */
[disabled] {
    opacity: 0.5;
}

/* Точное значение */
[type="email"] {
    border-color: blue;
}

/* Начинается с */
[href^="https"] {
    color: green;
}

/* Заканчивается на */
[src$=".pdf"] {
    background-image: url('pdf-icon.png');
}

/* Содержит */
[class*="btn"] {
    cursor: pointer;
}
```

## 🎨 Цвета в CSS

### Способы задания цвета

| Способ | Пример | Описание |
|--------|--------|----------|
| Ключевое слово | `red` | 147 предопределённых цветов |
| HEX | `#ff0000` | Шестнадцатеричный (RGB) |
| HEX (сокращённый) | `#f00` | Если символы повторяются |
| RGB | `rgb(255, 0, 0)` | Красный, зелёный, синий (0-255) |
| RGBA | `rgba(255, 0, 0, 0.5)` | RGB + прозрачность (0-1) |
| HSL | `hsl(0, 100%, 50%)` | Тон, насыщенность, светлота |
| HSLA | `hsla(0, 100%, 50%, 0.5)` | HSL + прозрачность |

### Примеры

```css
/* Именованные цвета */
.error { color: red; }
.success { color: green; }
.warning { color: orange; }

/* HEX — самый распространённый */
.primary { color: #3498db; }
.secondary { color: #2ecc71; }

/* RGB — удобно для программного изменения */
.card { background-color: rgb(245, 245, 245); }

/* RGBA — с прозрачностью */
.overlay { background-color: rgba(0, 0, 0, 0.7); }

/* HSL — удобно для подбора оттенков */
.theme-light { background-color: hsl(210, 50%, 95%); }
.theme-dark { background-color: hsl(210, 50%, 15%); }
```

### Свойства цвета

```css
.element {
    color: #333;              /* Цвет текста */
    background-color: #fff;   /* Цвет фона */
    border-color: #ccc;       /* Цвет рамки */
}
```

## 🔤 Работа с текстом и шрифтами

### Семейство шрифтов

```css
body {
    /* Список шрифтов: от предпочтительного к запасному */
    font-family: 'Open Sans', Arial, sans-serif;
}

/* Типы запасных шрифтов (generic) */
/*
    serif       — с засечками (Times New Roman)
    sans-serif  — без засечек (Arial)
    monospace   — моноширинный (Courier)
    cursive     — рукописный
    fantasy     — декоративный
*/
```

### Подключение веб-шрифтов (Google Fonts)

```html
<!-- В <head> HTML -->
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
```

```css
body {
    font-family: 'Roboto', sans-serif;
}
```

### Размер шрифта

```css
.text {
    /* Абсолютные единицы */
    font-size: 16px;     /* Пиксели */
    font-size: 12pt;     /* Пункты (для печати) */
    
    /* Относительные единицы (рекомендуется) */
    font-size: 1rem;     /* 1 rem = размер шрифта корневого элемента */
    font-size: 1.2em;    /* 1 em = размер шрифта родителя */
    font-size: 120%;     /* Проценты от родителя */
    
    /* Ключевые слова */
    font-size: small;
    font-size: medium;
    font-size: large;
}
```

### Насыщенность и стиль

```css
.text {
    /* Насыщенность (жирность) */
    font-weight: normal;     /* 400 */
    font-weight: bold;       /* 700 */
    font-weight: 300;        /* Тонкий */
    font-weight: 600;        /* Полужирный */
    
    /* Стиль */
    font-style: normal;
    font-style: italic;      /* Курсив */
    font-style: oblique;     /* Наклонный */
}
```

### Оформление текста

```css
.text {
    /* Выравнивание */
    text-align: left;
    text-align: center;
    text-align: right;
    text-align: justify;     /* По ширине */
    
    /* Декорации */
    text-decoration: none;           /* Убрать */
    text-decoration: underline;      /* Подчёркивание */
    text-decoration: line-through;   /* Зачёркивание */
    
    /* Трансформация */
    text-transform: uppercase;   /* ЗАГЛАВНЫЕ */
    text-transform: lowercase;   /* строчные */
    text-transform: capitalize;  /* Каждое Слово */
    
    /* Высота строки */
    line-height: 1.5;    /* Множитель (рекомендуется) */
    line-height: 24px;   /* Абсолютное значение */
    
    /* Межбуквенный интервал */
    letter-spacing: 2px;
    letter-spacing: -1px;
    
    /* Межсловный интервал */
    word-spacing: 5px;
}
```

## ⚖️ Каскадность и специфичность

### Каскадность

CSS означает "Cascading" — каскадные. Стили применяются **каскадом** сверху вниз.

```css
/* Если свойство задано дважды, побеждает последнее */
p {
    color: blue;
}

p {
    color: red;  /* Этот победит */
}
```

### Специфичность

Когда есть конфликт, браузер выбирает правило с **большей специфичностью**.

**Система подсчёта: (a, b, c, d)**

| Категория | Вес | Примеры |
|-----------|-----|---------|
| Инлайн-стили | (1,0,0,0) | `style="..."` |
| ID | (0,1,0,0) | `#header` |
| Классы, атрибуты, псевдоклассы | (0,0,1,0) | `.card`, `[type]`, `:hover` |
| Элементы, псевдоэлементы | (0,0,0,1) | `p`, `::before` |

**Примеры расчёта:**

```css
p { }                    /* (0,0,0,1) */
.card { }                /* (0,0,1,0) */
p.card { }               /* (0,0,1,1) */
#main .card { }          /* (0,1,1,0) */
#main .card p { }        /* (0,1,1,1) */
```

### !important

```css
.text {
    color: blue !important;  /* Побеждает почти всегда */
}
```

> **Важно:** Избегайте `!important`! Это признак плохой архитектуры CSS. Используйте только в крайних случаях.
